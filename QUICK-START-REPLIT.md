# 🚀 Build Rápido - Texturas APK (Replit)

## Link Direto: https://replit.com

## 3 Passos Simples:

### 1️⃣ Importe o Projeto
- Clique em **Create** → **Import from GitHub**
- Cole: `https://github.com/phmello98-create/texturas`
- Clique em **Import**

### 2️⃣ Execute o Build
No terminal do Replit (abaixo do editor), cole e aperte Enter:

```bash
./build-replit.sh
```

### 3️⃣ Baixe o APK
- No painel esquerdo, vá em: `app/build/outputs/apk/debug/`
- Clique com botão direito em `app-debug.apk` → **Download**

---

## 💡 Dicas

- **Tempo total**: 5-10 minutos
- **100% grátis**: Não pede cartão de crédito
- **Espaço**: ~3GB para SDK + dependências

## ❓ Erro Comum

Se aparecer "command not found", execute primeiro:

```bash
cd /home/runner/texturas
```

Depois execute novamente: `./build-replit.sh`

---

✅ **Pronto!** Baixe o APK e instale no seu Android!
